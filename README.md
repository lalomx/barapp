# barapp
Codigo fuente de la aplicaicon bar app

# Requisitos previos

- Instalar IDE Visual Studio Code
- Instalar docker
- Instalar node js version 10+

# Inicializar docker

1. Abrir el proyecto en el visual studio
2. Abrir una terminal
3. ejecutar docker-compose up -d
